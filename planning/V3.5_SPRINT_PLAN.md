# V3.5 Sprint Plan - Autonomous Agent Execution Guide

**Created:** 2026-01-14
**Target:** Complete V3.5 feature set before V4.0 remote deployment work

---

## Executive Summary

V3.5 organizes work into autonomous-friendly sprints that agents (orchestrator, the-drone, cli-agent/gemini) can execute with minimal human intervention. Each sprint is scoped for 1-2 days of work with clear entry/exit criteria.

### Priority Order
1. **P1 (Critical Path):** Remote Ingest Watcher, Embedded Chat Prototype
2. **P2 (High Value):** Full Chat Implementation, Langfuse, UX/Test Cleanup
3. **P3 (Nice to Have):** Content Calendar, Large Transcript Opt, Plugin Foundation
4. **P4 (Polish):** Google Docs Export, Project Rename
